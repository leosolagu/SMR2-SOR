### Instalar Active directory
En administrar nos saldrán varias opciones, pero, la que nos interesa es agregar roles y caracteristicas. (dentro de administrador de servidor)
<img width="779" height="560" alt="ACTIVEDIRETORY1-4" src="https://github.com/user-attachments/assets/2fc71793-64f1-4ec0-acd4-d468f011121a" />

Le damos a instalar caracteristicas y roles.

<img width="1032" height="774" alt="ACTIVEDIRETORY1-5" src="https://github.com/user-attachments/assets/a7e165e5-d6f7-4fbc-94b5-3ebc0f3e00cf" />

Selecionamos nuestro servidor (equipo)

<img width="1030" height="773" alt="ACTIVEDIRETORY1-6" src="https://github.com/user-attachments/assets/a5337d76-f3d5-4313-8090-8751f1a3ffb3" />

Y le clicamos a Servicios de dominio de Active Directory

Esperamos a que se instale, y posteriormente, en el apartado de herramienteas nos aparecerá la opción de "Administración de directivas y grupo"


<img width="1029" height="775" alt="activediretory" src="https://github.com/user-attachments/assets/f5a3afce-315e-4d43-9907-90ff0ea30d0b" />

Nos aparecera esta celda y pondremos "miservidor.net" en el nombre de dominio, siguiente.

<img width="1029" height="775" alt="ACTIVEDIRETORY1-1" src="https://github.com/user-attachments/assets/800059f1-ef66-41f1-a533-472c430de26d" />

Por medidas de seguridad le podemos configurar una contraseña. Tendrá que tener una mayúscula, minúscula y números

<img width="1036" height="766" alt="ACTIVEDIRETORY1-2" src="https://github.com/user-attachments/assets/b0f34fde-011b-4836-bcc7-15bb646ac23b" />

Le configuraremos un nombre "MISERVIDOR" es el mio.

<img width="1035" height="777" alt="ACTIVEDIRETORY1-3" src="https://github.com/user-attachments/assets/ce9d50e2-ee15-4494-8c65-b89edeb170c5" />

Y le configuraremos una ruta de acceso para la base de datos de AD DS, archivo de registro y SYSVOL.

Ahora ya tenemos instalado el Active directory, se nos reiniciara el ordenador, posteriormente, cuando lo inicies puedes observar que puedes elegir otro usuario.
